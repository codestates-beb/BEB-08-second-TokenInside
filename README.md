# Web3 Incentivized Community Project

Web3 커뮤니티 활동을 하면 토큰을 주는 형식으로 프로젝트.  

게시글을 작성하면 토큰을 받으며, 토큰을 활용해 NFT를 민팅 할 수 있고 토큰을 타인에게 전송도 가능합니다.
